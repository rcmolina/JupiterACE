Ace manual Code Examples (AMCE) Jupiter ACE 


In this archive file are the code examples from the Jupiter Ace's manual.
 When the contents of the zip file are extracted you will find a number of folders, eg. 
'Chapter 21' all the AceFORTH code examples from chapter 21 of the manual will be in this folder.
 Some chapters of the manual will not have any code examples the ones that do are 
chapters 8, 11, 12, 13, 15, 16, 17, 19, 20, 21, 23, 24, 25, and 26. Also in the folders
 are some screen dumps when the example shows a result output to the screen, see screen 
shots below tzx files for each example have been made.
